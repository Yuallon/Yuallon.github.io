
 今天中午的时候，大伯家的二姐带着她儿子来找我补习英语，小孩子已经读到了五年级。说是英语相对其他功课而言很是不好，希望我教他读一下学过的英文单词。

一番交谈过后，我明白他不会读音标。准确来讲，单词和音标是分不清的。就是听了别人读一下发音，自己跟着读一下，然后硬生生的把汉语含义记住。不知是他没有好好学习音标发音呢？还是老师没有系统的去讲这部分内容呢？

不管如何，既然想顺利的去读单词。音标学习可谓是最最基本的东西。自己就在B站搜索了一个音标发音视频 [英语48个国际音标读音](https://www.bilibili.com/video/BV1UE411P7um?t=95) ，也顺便温习一下这方面的知识。

第一天学习，只教了孩子学习20个音标发音，有6个元音和14个辅音。如下

| 元音 | /i:/, /e/, /u:/, /ɔː/, /ɑː/, /ɜː/                            |
| ---- | ------------------------------------------------------------ |
| 辅音 | /p/, /t/, /k/, /f/, /ʃ/, /s/, /b/, /d/, /g/, /z/, /h/, /l/, /m/, /n/ |

从近12点开始学习，一直学到了下午3点半。中间也让他写了会儿数学，英语作业。

其实看起来是我教孩子英语发音，其实也是自己详细的去过一遍这个最基本的知识点。想起来，自己也很久没有认真学习英文了。自己的博客基本功能也有了，是该开始每天英文的学习了。

还是和以前一样，每天一遍英文新闻，读完之后自己写内容总结，和单词总结。然后保证每天最少有1个小时的英文听力，自己也要时不时的口语一下。

2020年6月21号 于家中。
