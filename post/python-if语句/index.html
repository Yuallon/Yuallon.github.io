#if语句
if 判断条件1:
  	执行语句1
elif 判断条件2:
  	执行语句2
else:
 		执行语句3

注：其中 elif 语句可以有多个，对于 if 和 elif 中的判断条件，运行第一个给出True值后面的执行语句，且运行后不再判断后面 elif语句。如果最后一个 elif 语句返回的仍是False，则执行 else 中的语句。

语法解读：

python中的 if 条件语句 通过依次执行语句中的判断条件，如果判断结果返回的是True，则运行紧跟后面的执行语句。如果判断条件1返回的值是True，则不会再运行后面 elif，else语句。如果判断条件都为False，则会执行else中的语句。

if 语句可以嵌套！
