###变量名命名方法

要求：

  1. 见名知其意
  2. 由字母、下划线、数字组成，不能以数字开头
  3. 区分大小写
  4. 不能和关键字重复

方法：

  • 小驼峰命名方法：myName
  • 大驼峰命名方法：每个单词第一个字母都大写, userNameFlag
  • 下划线：send_name
